CHILUKAMARI 2017 A computational model of visual attention
==========================================================

This dataset is supplementary to the full-text thesis. It 
consists of two folders, each containing a separate README 
file. There is also this README document for the dataset 
as a whole, and a coversheet with further information.

The folder 'Evaluation metrics' contains four MATLAB files 
in .m format and one plain text README file.

The folder 'Visual Saliency Model (MATLAB)' contains seven 
MATLAB files in .m format, seven image files in .jpg/.jpeg 
format, three image files in .bmp format and one plain text 
README file.
